# Люба (@asluba_bot)

Живой собеседник в Telegram — общается в личке и **активно ведёт себя в группах
и комментариях к каналам**. Широкие знания о мире, человеческие реакции,
динамическое настроение, понимание времени и пространства.

Работает на **локальной модели** (RuadaptQwen3-4B, как Ася) + бесплатные
облачные провайдеры (Pollinations free — без ключа; опционально Groq / Gemini /
OpenRouter / Cloudflare / HuggingFace, если добавить ключи).

## Что умеет

- 💬 **Личные чаты** — общается как знакомая, помнит контекст разговора.
- 👥 **Группы** — отвечает всем, кого упомянули/ответили, и **проактивно
  комментирует** сообщения (настройка вероятности). Понимает, КТО пишет, ГДЕ
  и на что отвечает. Помнит недавние темы и факты о людях в чате.
- 📰 **Каналы** — короткие живые комментарии к постам (как активный подписчик).
- 🖼 **Картинки** — видит и понимает фото (VLM через Pollinations free).
- 🌐 **Проверка в интернете** — перепроверяет факты/новости/цены (DuckDuckGo).
- 🔗 **Партнёрские программы** — берёт тот же источник, что Ася
  (`sochiautoparts.ru/partners.json`), и естественно вплетает ссылки в диалог
  и комментарии, когда это к месту.
- 😊 **Настроение** — динамическое: зависит от времени суток, случайного
  дрейфа и тональности беседы.
- ⏰ **Время и пространство** — знает текущую дату/время по Москве, время суток,
  день недели, сезон; живёт в Сочи.
- 🚫 **Без политики и войны** — мягко уходит от таких тем.

## Принцип работы (отличие от Аси)

Ася ведёт автоканал и делает посты. **Люба не ведёт каналы** — её задача
общаться с людьми: в личке и активно в группах/комментариях.

## Архитектура AI (failover)

```
Уровень 0: Локальная модель (RuadaptQwen3-4B GGUF, CPU)  ← chat & comment
Уровень 1: Опциональные облака (Groq → OpenRouter → Gemini → CF → HF)  ← если есть ключи
Уровень 2: Pollinations FREE (без авторизации — всегда доступен)
Запас:     Статические фразы
```

- Чаты и комментарии идут через **локальную модель первой** (экономия баланса
  облака + приватность: сообщения из групп не покидают машину).
- Vision — через Pollinations free (локальная модель не умеет картинки).

## Запуск

### Локально

```bash
pip install -r requirements.txt
# скачать модель RuadaptQwen3-4B-Instruct Q4_K_M (~2.3GB) в models/
cp .env.example .env  # заполнить BOT_TOKEN
python -m bot.main
```

### 24/7 через GitHub Actions

В репозитории уже есть `.github/workflows/bot.yml`:
- cron каждые 5 часов + self-dispatch → бот работает круглосуточно;
- кэширует модель и БД;
- коммитит БД в git для сохранения памяти между запусками;
- unlimited auto-restart при падениях.

**Нужные секреты** (Settings → Secrets and variables → Actions):

| Секрет | Обязательно | Описание |
|---|---|---|
| `BOT_TOKEN` | ✅ | токен @asluba_bot |
| `OWNER_ID` | ✅ | Telegram ID владельца (265070804) |
| `GH_PAT_TOKEN` | ✅ | GitHub PAT для self-dispatch |
| `HF_TOKEN` | желательно | для скачивания модели (gated) |
| `GROQ_API_KEY` | опц. | бесплатный быстрый Llama 70B |
| `GEMINI_API_KEY` | опц. | бесплатный Gemini |
| `OPENROUTER_API_KEY` | опц. | бесплатные модели |
| `CF_ACCOUNT_ID_1` / `CF_API_TOKEN_1` | опц. | Cloudflare Workers AI |
| `POLLINATIONS_API_KEY` | опц. | лучше rate-limit у Pollinations |

Без опциональных ключей бот всё равно работает: локальная модель + Pollinations free.

## ⚠️ Важно: Privacy Mode для групп

Чтобы Люба получала **все** сообщения в группах (а не только упоминания и ответы),
отключите Privacy Mode у бота:

1. Откройте `@BotFather`
2. `/mybots` → `@asluba_bot` → **Bot Settings** → **Group Privacy** → **Turn off**

После этого добавьте Любу в группы/каналы (в каналы — как админа с правом постить
комментарии).

## ⚠️ Важно: настройки для полной активности

### 1. Privacy Mode для групп (чтобы бот видел ВСЕ сообщения)
1. `@BotFather` → `/mybots` → `@asluba_bot` → **Bot Settings** → **Group Privacy** → **Turn off**
2. После этого Люба будет получать все сообщения в группах, а не только упоминания.

### 2. Реакции / лайки в каналах (чтобы бот ставил 👍❤️🔥)
Чтобы Люба ставила реакции на посты канала, она должна быть **админом канала**:
1. Открой канал → **Manage Channel** → **Administrators** → **Add Administrator**
2. Найди `@asluba_bot` и добавь
3. Дай права: минимум **Post Messages** (для комментариев) + реакции работают автоматически
4. Теперь Люба будет реагировать (👍❤️🔥😄😮🙏) на ~50% постов и писать комментарии

В группах реакции работают без админки — Люба просто должна быть участником.

## Режимы работы AI (оптимизация скорости)

- **Личные чаты**: локальная модель (быстро, ~2-5с) → cloud fallback
- **Группы**: локальная модель (приватность — сообщения не покидают машину) → cloud
- **Vision (фото)**: только в личных чатах. В группах vision ОТКЛЮЧЕН для экономии ресурсов — Люба ориентируется на текст событий и подписи к фото.
- **Комментарии**: локальная модель → Pollinations free (GET-endpoint сначала — надёжнее)
- **Объём ответов**: адаптивный — короткие реакции на короткие сообщения, развёрнутые на вопросы/обсуждения

## Структура

```
bot/
  config.py          — конфиг + личность Любы (промпты, знания о мире)
  database.py        — SQLite (память, группы, каналы, настроение, кэш)
  partners.py        — партнёрские программы (sochiautoparts.ru/partners.json)
  web_search.py      — веб-поиск (DuckDuckGo) для проверки фактов
  media_handler.py   — загрузка фото → base64 для vision
  mood.py            — динамическое настроение
  context.py         — сбор контекста «кто/что/где»
  main.py            — точка входа
  handlers/
    chat.py          — личные чаты
    groups.py        — активное поведение в группах
    channels.py      — комментарии к постам каналов
    admin.py         — команды владельца
ai/
  router.py          — failover-маршрутизация
  providers/
    local_provider.py        — RuadaptQwen3-4B (llama-cpp-python)
    pollinations_provider.py — Pollinations free (текст + vision)
    optional_providers.py    — Groq/OpenRouter/Gemini/Cloudflare/HF
```

## Команды

- `/start`, `/help` — приветствие
- `/clear` — забыть историю чата
- `/mood` — показать текущее настроение
- `/stats` (владелец) — статистика запросов по провайдерам
- `/reload_partners` (владелец) — обновить партнёрские программы
- `/channel_on <id>` / `/channel_off <id>` (владелец) — вкл/выкл комментарии канала
- `/broadcast <chat_id> <text>` (владелец) — отправить сообщение
